Listas

Las listas en HTML sirven para listar contenido. En función del tipo de contenido de la lista hay tres tipos de listas:

  • unordered list <ul>: Se utilizan cuando el orden de los elementos no influye.
  • ordered list <ol>: Se utilizan cuando el orden de los elementos es importante.
  • definition list <dl>: Se utilizan para hacer una lista de definiciones.
    • Definition term <df>: El término que se va a definir.
    • Definition description <dd>: La descripción del término.

Cada elemento de la lista se representa con la etiqueta <li>, tanto en las ul como en las ol.

Atributos de las listas

  • <ol>

    • type: Estilo de numeración (1, A, a, I, i)
    • start: Inicio de las secuencia (un número)

  • <ul>

    • type: Estilo de los items (disc, square, circle)
