Metadatos

El elemento head de un documento HTML es la parte que no se muestra en el navegador cuando se carga la página. Contiene información como el título (<title>) de la página, enlaces al CSS (si quieres aplicar estilo a tu contenido HTML con CSS), enlaces para personalizar favicon, y otros metadatos (datos sobre el HTML, como quién lo escribió y palabras claves importantes que describen el documento).

  • <title>: Se usa para poner un título al documento.
  • <meta>
    • charset: Este elemento especifica la codificación de caracteres del documento. utf-8 es un conjunto de caracteres universal que incluye casi todos los caracteres de casi cualquier idioma humano.
    • name: especifica el tipo de metadato del que se trata; es decir, qué tipo de información contiene.
      • author: Autor de la página o sitio web.
      • description: Descripción de esta (la cual usualmente es la que es visible debajo del link de acceso al buscar páginas en Google o otros motores de búsqueda).
    • content: especifica el contenido del metadato en sí.
    • Open Graph Data: es un protocolo que permite que cualquier página web se convierta en un objeto enriquecido en un gráfico social.
      • og:title: El título de objeto como debería aparecer dentro del gráfico.
      • og:type: El tipo de objeto, por ejemplo, "video.movie". Dependiendo del tipo que se especifique, es posible que también se requieran otras propiedades.
      • og:image: Una URL de imagen que debe representar el objeto dentro del gráfico.
      • og:url: La URL canónica del objeto que se utilizará como su ID permanente en el gráfico.
      • og:description: Una descripción de una o dos frases del objeto.
  • <link>: Especifica la relación entre el documento actual y un recurso externo, usualemte son archivos CSS y favicons. Generador favicons
  • <script>: Especifica la relación entre el documento actual y un recurso de JavaScript.
  • <html lang="es">: El atributo lang en la etiqueta de apertura <html> sirve para especificar el idioma de la página (en este caso español).
