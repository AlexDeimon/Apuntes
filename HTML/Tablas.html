Tablas <table>

Las tablas en HTML sirven para mostrar contenido tabulado.

La estructura básica de una tabla se compone de:

  1. <table>: etiqueta que encierra una tabla.
  2. table row <tr>: etiqueta que construye una fila.
  3. table data <td>: etiqueta que construye una celda.

Los titulos de las tablas se establecen con la etiqueta <caption>, es una etiqueta opcional, y según la especificación esa etiqueta se coloca justo despues de la etiqueta <table>

Las cabeceras de las tablas se establecen con la etiquera <thead>.

Dentro se tiene una etiqueta <tr> normal, pero en el caso de las celdas se establecen con <th> en lugar de <td>

El contenido de la tabla debe ir dentro de una etiqueta <tbody> para representar el cuerpo de la tabla.

De forma opcional se puede colocar una etiqueta <tfoot> a la tabla para establecer un pie de tabla.

Para hacer que las celdas ocupen más de una fila o más de una columna hay 2 atributos:

  • rowspan: sirve para que una celda ocupe mas de una fila
  • colspan: sirve para que una celda ocupe mas de una columna

Cuando se necesita seleccionar una columna, existe la etiqueta <colgroup>, que permite seleccionar una columna en concreto. Dentro se pone tantas etiquetas col como columnas se tengan, cada etiqueta col equivaldrá a una columna siguiendo el mismo orden que tienen en la tabla.

si se necesita que la etiqueta col agrupe mas de una columna, está el atributo span, que funciona igual que rowspan y colspan
