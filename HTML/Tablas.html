Tablas <table>

Las tablas en HTML sirven para mostrar contenido tabulado.

La estructura básica de una tabla se compone de:

<table>:
Etiqueta que encierra una tabla.
table header <th>:
Etiqueta que contruye las celdas de la cabecera de la tabla.
table row <tr>:
Etiqueta que construye una fila.
table data <td>:
Etiqueta que construye una celda.

Hay un grupo de etiquetas para organizar el contenido de la tabla, estas son:

<caption>:
Establece el título de la tabla.
<thead>:
Establece la cabecera de la tabla.
<tbody>:
Establece el cuerpo de la tabla.
<tfoot>:
Establece el pie de la tabla.

Para hacer que las celdas ocupen más de una fila o más de una columna hay 2 atributos:

rowspan:
Sirve para que una celda ocupe mas de una fila.
colspan:
Sirve para que una celda ocupe mas de una columna.

Cuando se necesita seleccionar una columna, existe la etiqueta <colgroup>, que permite seleccionar una columna en concreto. Dentro se pone tantas etiquetas <col> como columnas se tengan, cada etiqueta col equivaldrá a una columna siguiendo el mismo orden que tienen en la tabla.

Si se necesita que la etiqueta <col> agrupe más de una columna, está el atributo span, que funciona igual que rowspan y colspan.
