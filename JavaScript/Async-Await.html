Async/Await

Las palabras clave async/await, son una forma de gestionar las promesas de una forma más sencilla. Con async/await seguimos utilizando promesas, pero abandonamos el modelo de encadenamiento de .then() para utilizar uno en el que trabajamos de forma más tradicional.

Async:

La palabra clave async define a una función como una función asíncrona, por lo que el código no se ejecutará de forma secuencial y automaticámente la función va a devolver una promesa.

Await:

Establece bloqueos en el código hasta la operación posterior a la palabra clave se haya resuelto. Await no funcionará si no se esta dentro de una función async.
