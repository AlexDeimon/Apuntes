Eventos

Un evento es cualquier cosa que puede suceder en el documento.

  • El contenido se ha leido.
  • El contenido se ha cargado.
  • El usuario mueve el ratón.
  • El usuario pulsa una tecla.
  • El usuario hace click.
  • Entre muchos otros.

Eventos del ratón:

click:
Cuando se pulsa el botón izquierdo del ratón.
dblclick:
Cuando se pulsa dos veces seguidas el botón izquierdo del ratón.
mouseenter:
Cuando se hace hover en zona que tiene el evento.
mouseleave:
Cuando se deja de hacer hover en zona que tiene el evento.
mousedown:
Cuando se pulsa o se mantiene pusado el boton izquierdo del ratón.
mouseup:
Cuando se suelta el boton izquierdo del ratón.
mousemove:
Cuando se mueve el ratón en zona que tiene el evento.

Eventos del teclado:

keydown:
Cuando se pulsa una tecla.
keyup:
Cuando se suelta una tecla.
keypress:
Cuando se pulsa una tecla y no se suelta.
