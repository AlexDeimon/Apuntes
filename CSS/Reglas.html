Reglas-at en CSS

Las reglas-at son un tipo de regla que instruyen a CSS sobre cómo comportarse. Comienzan con un signo de arroba, '@', seguido de un identificador e incluyen todo hasta el siguiente punto y coma, o el siguiente bloque CSS.

Lista de reglas-at:

Regla-at Descripción Mejor explicada
@Counter-Style Define estilos de para utilizar en contadores CSS que no forman parte del conjunto predefinido de estilos. @counter-Style
@font-face Describe el aspecto de una fuente externa que se va a descargar. @font-face
@font-feature-values Activa/desactiva características especiales en una tipografía. @font-feature-values
@import Le dice al motor CSS que incluya una hoja de estilo externa. @import
@keyframes Describe el aspecto de los pasos intermedios en una secuencia de animación CSS. @keyframes
@media Una regla de grupo condicional que aplicará su contenido si el dispositivo cumple con los criterios de la condición definida mediante una media query. Usado para responsive. @media
@page Describe el aspecto de los cambios de diseño que se aplicarán al imprimir el documento. @page
@supports Una regla de grupo condicional que aplicará contenido si el navegador cumple con los criterios de la condición dada. @supports
