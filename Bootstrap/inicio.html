¿Qué es Boostrap?

Bootstrap es un framework de CSS que facilita y estandariza el desarrollo de sitios web. Esta pensando para que se adapte tanto a las pantallas de equipos de escritorio como a móviles y tablets.

La versión estable actual es la 4.x

Hay 2 formas de trabajar con Boostrap:

  1. Descargar archivo CSS del sitio http://getbootstrap.com.
  2. Utilizar un servidor (CDN) donde se alojan todos los archivos de Bootstrap. Esto mediante una etiqueta <link>
    • Muchos de los componentes requieren el uso de JavaScript para funcionar. Específicamente, requieren jQuery, Popper.js y los propios complementos JavaScript de Bootstrap. Se deben añadir los siguientes <script>, para habilitarlos. jQuery debe venir primero, luego Popper.js y luego los complementos de JavaScript.
