List Group

List Group son un componente flexible y potente para mostrar una serie de contenidos.

  • HTML
  • CSS
  • JavaScript
  • Este List Group esta conformado por un <ul> con la clase list-group.
  • cada <li> posee la clase list-group-item.

Links & botones:

HTML CSS JavaScript
  • Otra forma de crear un List Group es mediante un <div> con la clase list-group.
  • Dicho div puede contener <a> o <button> como items de la lista agregando las clases list-group-item y list-group-item-action.

Items activos y deshabilitados:

  • HTML
  • CSS
  • JavaScript
  • Bootstrap
  • El item HTML tiene la clase active.
  • El item Bootstrap tiene la clase disabled.
