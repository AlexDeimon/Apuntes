Collapse

La componente collapse permite mostrar u ocultar el contenido de un "div" cuando se presiona un botón o ancla.

Para su funcionamiento requiere importar las librerías de Javascript.

La clase involucrada para su funcionamiento se llama "collapse".

Múltiples objetivos:

Un <button> o <a> puede mostrar y ocultar varios elementos al hacer referencia a ellos con un selector JQuery en su href o atributo data-target. Múltiples <button> o <a> pueden mostrar y ocultar un elemento si cada uno hace referencia a él con su href o atributo data-target.
